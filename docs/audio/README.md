# `StuxnetHN.Audio`

# IMPORTANT
`Stuxnet.Audio` is *not* included with Stuxnet by default. To use anything here, make doubly sure that you have *both* `StuxnetHN.dll` and `StuxnetHN.Audio.dll` in your extension's plugins folder!
